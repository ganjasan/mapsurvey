# Participedia

> A global crowdsourcing platform for researchers, educators, practitioners, policymakers, and activists interested in public participation and democratic innovations. Documents participatory governance methods from 160 countries.

**Source:** [Participedia](https://participedia.net/)

---

## Platform Overview

Participedia enables users to explore, contribute, and teach using a comprehensive database of democratic participation examples worldwide. The platform serves as a knowledge-sharing hub for how communities implement participatory governance methods.

### Key Statistics

| Content Type | Count |
|--------------|-------|
| Cases | 2,367 |
| Methods | 386 |
| Organizations | 875 |
| Collections | 20 |
| Countries | 160 |

## Content Types

### Cases
Documented implementations of specific participation methods in real-world contexts.

- **Type:** Case studies
- **Related:** [[participatory-budgeting]], [[citizens-assemblies]]

### Methods
Descriptions of participatory processes and democratic innovations.

- **Examples:** Citizens' juries, deliberative polling, participatory budgeting
- **Type:** Methodology reference
- **Related:** [[deliberative-democracy]], [[civic-engagement-methods]]

### Organizations
Profiles of groups designing and implementing civic innovations.

- **Type:** Directory
- **Related:** [[civic-tech-organizations]], [[democracy-practitioners]]

### Collections
Thematic groupings of cases and methods.

- **Type:** Curated content
- **Related:** [[thematic-research]]

## Platform Features

### Search & Discovery
- Advanced search syntax (AND, OR, NOT, quotes, parentheses)
- AI search assistant (beta) for finding innovations
- Bookmarking system for logged-in users

### Contribution Tools
- **Quick Submit** - rapid entry creation
- Draft auto-save functionality
- Approval workflow for new entries

### Multilingual Support
Available in 8 languages:
- English, French, German, Spanish
- Chinese, Italian, Portuguese, Dutch

Machine translation for open text fields; fixed data fields sync across languages.

### Education
- Teaching resources
- Participedia School program
- **Related:** [[participatory-mapping-education]]
