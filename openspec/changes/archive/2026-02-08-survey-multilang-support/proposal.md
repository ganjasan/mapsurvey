## Why

Опросы часто проводятся среди респондентов, говорящих на разных языках. Сейчас весь контент опроса (названия секций, тексты вопросов, варианты ответов) хранится только на одном языке. Чтобы охватить аудиторию на нескольких языках, приходится создавать отдельные дублирующие опросы. Мультиязычная поддержка позволит одному опросу обслуживать респондентов на разных языках без дублирования структуры и данных.

## What Changes

- Новый экран выбора языка — первое, что видит пользователь при открытии мультиязычного опроса
- Переводы контента опроса: заголовки секций (`title`, `subheading`), тексты вопросов (`name`, `subtext`), варианты ответов (`OptionChoice.name`)
- Хранение переводов в отдельных моделях, связанных с оригинальными сущностями
- Автоматическое переключение языка интерфейса (Django i18n) при выборе языка респондентом
- Админ-интерфейс для управления переводами
- Опросы без переводов работают как раньше (backwards compatible)

## Capabilities

### New Capabilities

- `survey-content-translation`: хранение и получение переводов контента опроса (секции, вопросы, варианты ответов) на разные языки
- `survey-language-selection`: экран выбора языка для респондента и сохранение выбора в сессии

### Modified Capabilities

- `ui-internationalization`: расширение для автоматического переключения языка интерфейса на основе выбора респондента в опросе
- `survey-serialization`: расширение импорта/экспорта для сохранения и восстановления переводов контента

## Impact

**Модели данных:**
- Новые модели для хранения переводов: `SurveySectionTranslation`, `QuestionTranslation`, `OptionChoiceTranslation`
- Новое поле в `SurveyHeader` для списка доступных языков
- Сохранение выбранного языка в `SurveySession`

**Views:**
- Новый view для страницы выбора языка
- Модификация существующих views для подстановки переведённого контента

**Шаблоны:**
- Новый шаблон страницы выбора языка
- Модификация шаблонов секций для отображения переведённого контента

**Формы:**
- Модификация `SurveySectionAnswerForm` для использования переведённых текстов

**Админка:**
- Inline-формы для редактирования переводов в существующих моделях

**URL-маршрутизация:**
- Новый URL-паттерн для страницы выбора языка

**Сериализация:**
- Экспорт/импорт `available_languages` в survey.json
- Экспорт/импорт переводов секций, вопросов, вариантов ответов
- Экспорт выбранного языка сессии в responses.json
